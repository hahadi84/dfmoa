## 현재 스케줄

- `0 15 * * *` UTC: 매일 00:00 KST
- `0 3 * * *` UTC: 매일 12:00 KST

하루 2회 실행을 유지합니다. 수집 대상이 40개로 늘었지만 롯데와 신세계는 정책상 자동 재시도하지 않도록 변경되어 신라 요청량은 과도하지 않은 수준입니다.

## 배포 페이지 신선도

2026-04-21 14:35 KST 기준 배포 홈에서 확인되는 최신 가격 기준 시각은 `2026-04-21 13:25`로 24시간 이내입니다.

## 실패 알림

이메일 실패 알림은 이번 변경에서 추가하지 않았습니다. Gmail 앱 비밀번호와 GitHub Secrets 설정이 필요하므로, 운영자가 알림 계정을 준비한 뒤 별도 세션에서 적용하는 편이 안전합니다.
