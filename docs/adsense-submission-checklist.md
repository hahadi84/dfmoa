# AdSense 심사 신청 전 체크리스트

## 도메인

- [ ] 커스텀 도메인 권장 (netlify.app 서브도메인은 승인율 낮음)
- [ ] HTTPS ✅

## 콘텐츠

- [ ] 색인 30+ 페이지 ✅ (149 URL)
- [ ] 각 페이지 본문 300자+
- [ ] 오리지널 콘텐츠

## 필수 페이지

- [ ] About · 연락처 · 개인정보처리방침 · 이용약관 (푸터 접근)

## 기술

- [ ] sitemap.xml 제출 ✅
- [ ] 404 페이지
- [ ] 모바일 반응형

## 수익화

- [ ] 쿠팡 제휴 `rel="sponsored nofollow"`
- [ ] 제휴 고지 명확 ✅

## 신청 절차

1. https://www.google.com/adsense/ 사이트 추가
2. AdSense 심사 코드 복사 → Codex에 전달 → `<head>` 삽입·배포
3. "검토 요청" 클릭 → 2일~2주 대기
4. 승인 후 ads.txt 업데이트 + 광고 슬롯 실제 배치
